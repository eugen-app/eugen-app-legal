# Политика конфиденциальности

Дата обновления: 17.02.2026

Nymira — это офлайн-развлекательное приложение. Мы уважаем вашу конфиденциальность и обязуемся защищать её.

## 1. Сбор данных

Приложение не собирает, не хранит и не передаёт персональные данные.

В приложении:
- нет аккаунтов
- нет трекинга
- нет аналитики
- нет рекламы
- нет внешних серверов

Все функции работают локально на вашем устройстве.

## 2. Датчики устройства

Приложение может использовать датчики движения для интерактивных функций. Эти данные обрабатываются только на устройстве и не передаются.

## 3. Сторонние сервисы

Приложение распространяется через Google Play. Google может обрабатывать технические данные в рамках своей платформы.

Политика конфиденциальности Google:
https://policies.google.com/privacy

Разработчик не имеет доступа к этим данным.

## 4. Дети

Приложение сознательно не собирает персональные данные детей.

## 5. Изменения

Политика конфиденциальности может обновляться. Новая версия будет опубликована с обновлённой датой.
